Аптечка для мамы и ребёнка

Что должно быть под рукой до и после родов

Безопасные препараты и средства ухода
Правила домашней аптечки
  • Храните лекарства в тёмном, сухом, прохладном месте (не в ванной, не на солнце).
  • Проверяйте срок годности каждые 3–6 месяцев.
  • Держите аптечку в недоступном для детей месте.
  • Имейте список лекарств с дозировками и показаниями.
  • Никогда не занимайтесь самолечением во время беременности и кормления! Даже безобидные на первый взгляд препараты могут быть опасны.
Важно: все препараты из этой статьи — только для ознакомления. Назначать их должен врач!
Аптечка для беременной
Фолиевая кислота (400–800 мкг/день) — до 12 недель
Йодомарин / Калия йодид (200–250 мкг/день) — при йододефиците
Препараты железа (при анемии) — по назначению врача
Витамин D3 (800–2000 МЕ/день) — для профилактики рахита у малыша
Но-шпа / Дротаверин — при спазмах (только по назначению)
Антациды (Ренни, Маалокс, Гастал) — при изжоге (разрешены)
Свечи с папаверином — при тонусе матки (по назначению)
Крем от растяжек — для профилактики стрий
Запрещено при беременности: аспирин, ибупрофен (в III триместре), большинство антибиотиков, снотворные, многие обезболивающие. Любой приём препаратов — только после консультации с врачом!
Аптечка для кормящей мамы
Парацетамол — жаропонижающее и обезболивающее (безопасен при ГВ)
Ибупрофен — после консультации с врачом (проникает в молоко в малых дозах)
Но-шпа — спазмолитик (безопасен при ГВ)
Спреи для горла (Гексорал, Тантум Верде) — разрешены при ГВ
Солевые растворы для носа (Аквамарис, Аквалор) — безопасны
Крем с декспантенолом (Бепантен) — для трещин сосков
Лактогонные средства (Апилак, чай с фенхелем) — по желанию
Валидол — при лёгкой тахикардии (безопасен)
При простуде: парацетамол + обильное питьё + солевые растворы в нос. Большинство капель от насморка (ксилометазолин) при ГВ не запрещены, но лучше коротким курсом.
Аптечка для новорожденного
Перекись водорода (3%) — для обработки пупка
Зеленка или хлорофиллипт (1%) — для пупка
Пипетка (2 шт.)
Стерильная вата, ватные диски, ватные палочки с ограничителем
Аспиратор (соплеотсос) — для носа
Солевой раствор для носа (Аквамарис бэби, физраствор) — от корочек
Крем под подгузник (цинковый) — от опрелостей
Присыпка (тальк/кукурузный крахмал) — альтернатива крему
Детское масло
Детский крем (увлажняющий)
Ножницы с закруглёнными концами или пилочка для ногтей
Градусник (электронный или бесконтактный)
Медицинский спирт (для обработки пупка по назначению врача)
Средства от колик и газов
Симетикон (Эспумизан L, Саб-Симплекс, Боботик) — снижает газообразование
Укропная вода / чай с фенхелем
Грелка для живота (или тёплая пелёнка)
Пробиотики (капли с Lactobacillus reuteri) — нормализуют микрофлору
Важно: перед применением любых препаратов новорождённому проконсультируйтесь с педиатром. Колики часто проходят сами к 3–4 месяцам.
Жаропонижающие для детей
Парацетамол (Панадол, Эффералган) — суспензия или свечи (с 1 мес.)
Ибупрофен (Нурофен) — с 3 мес. (действует быстрее и сильнее)
Детский градусник (лучше электронный или бесконтактный)
Важно: дозировка строго по весу! Не используйте анальгин и аспирин у детей — они опасны. При температуре выше 38,5°C у ребёнка до 3 месяцев — вызовите скорую.
Антигистаминные (от аллергии)
Фенистил капли — с 1 мес. (от зуда, крапивницы)
Зиртек / Цетрин капли — с 6 мес. (по назначению)
Лоратадин (Кларитин) — с 2 лет
При лёгкой аллергии у новорождённого часто достаточно исключить аллерген и использовать увлажняющие кремы.
Перевязочные материалы
Бинты стерильные — 2–3 шт.
Пластырь (обычный и гипоаллергенный)
Марлевые салфетки стерильные
Эластичный бинт — для компрессии после родов
Для обработки пупка новорождённого
  • Перекись водорода 3% — размягчает корочки
  • Зеленка 1% (бриллиантовый зелёный) — подсушивает
  • Хлорофиллипт 1% — альтернатива зеленке
  • Пипетка — для закапывания перекиси
  • Стерильные ватные палочки
Схема обработки: 1–2 раза в день. Капнуть перекись → подождать, пока зашипит → промокнуть сухой палочкой → обработать зеленкой.
Приборы и аксессуары
Электронный градусник
Бесконтактный инфракрасный термометр
Гигрометр (для влажности в комнате)
Увлажнитель воздуха (оптимально 50–60% влажности)
Сургутский клинический перинатальный центр — забота на каждом этапе
